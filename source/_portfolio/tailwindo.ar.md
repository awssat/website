## نظرة عامة

Tailwindo هي أداة تحويل تقوم بتحويل فئات Bootstrap CSS إلى معادلات Tailwind CSS عبر ملفات HTML/PHP. مع أكثر من **1,100 نجمة** على GitHub، أصبحت أداة أساسية للمطورين الذين ينتقلون من Bootstrap إلى Tailwind CSS.

## الميزات الرئيسية

- **التحويل التلقائي**: يحول فئات Bootstrap إلى Tailwind CSS في ملفات فردية، أو مقتطفات كود، أو مجلدات كاملة
- **بنية قابلة للتوسيع**: مصممة لإضافة دعم لأطر عمل CSS إضافية بخلاف Bootstrap
- **استخراج CSS**: يمكنه استخراج الأنماط المحولة كملفات CSS منفصلة باستخدام توجيه `@apply` في Tailwind
- **التوافق العكسي**: يحافظ على أسماء الفئات الأصلية للانتقال السلس
- **واجهة CLI**: أداة سطر أوامر سهلة الاستخدام مع توزيع Packagist

## المشكلة المحلولة

يتطلب الانتقال من Bootstrap إلى Tailwind CSS تقليدياً إعادة كتابة الفئات يدوياً في قاعدة الكود بأكملها. يُبسط Tailwindo هذه العملية من خلال إعادة هيكلة أوراق الأنماط تلقائياً، مما يوفر على المطورين ساعات لا تحصى من العمل الممل.
